基本原理

微信服务器就相当于一个转发服务器，终端（手机、PAD等）发起请求到微信服务器，微信服务器再讲请求转发到我们开发的服务，我么开发的服务器处理完毕之后，再回发给微信服务器，微信服务器再回复给终端

我们需要做的，就是对HTTP请求做出响应。

实现步骤

  • 云服务器购买
    微信公众号开发必须有服务器，或者使用内网映射工具。
  • 微信公众平台账号申请
    申请微信公众平台账号，并开启开发者模式，或者申请测试账号。本例为了不影响正式的公众号使用测试账号就行演示。
    测试账号申请地址：https://mp.weixin.qq.com/debug/cgi-bin/sandbox?t=sandbox/login
